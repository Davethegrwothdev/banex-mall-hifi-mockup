## STRATEGIC NOTE

The **big win here is emotional framing**:

The old site sells *space + mall features*.

The new PRD needs to sell **“What will I enjoy if I come there today?”**

That single shift will drive:

* longer session time
* more scroll depth
* more map clicks
* more direction taps
* more real foot traffic

Exactly what your KPI model wants.
